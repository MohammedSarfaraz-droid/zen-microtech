

# Zen Microtech — Premium Technology Website

## Design System
- **Theme**: Premium light luxury inspired by Apple × Stripe × Notion
- **Colors**: Ice White (#F8FAFC), Deep Electric Blue (#0F3D91), gradient accents (#0F3D91 → #2563EB → #60A5FA)
- **Typography**: Plus Jakarta Sans for headings, Inter for body text
- **Style**: Glassmorphism accents, soft shadows, rounded corners, subtle gradient overlays
- **Images**: AI-generated using Lovable's built-in image generation (tech repair labs, devices, server racks, team photos)

## Pages & Structure

### 1. Landing Page
- **Hero**: Full-screen with animated mesh gradient background, bold headline, dual CTAs, floating glass cards (Phone Repair, Networking, Security, Refurbished), AI-generated tech lab image
- **Services Overview**: 3 premium cards with hover lift animations and AI images — Tech Repair, Networking & Security, Refurbished Devices
- **Why Choose Us**: 6-point icon grid on subtle gray background
- **Process Timeline**: Horizontal 4-step flow (Consultation → Diagnosis → Repair → Delivery)
- **Refurbished Showcase**: Apple-store-style product grid with floating device effect, condition badges, prices
- **CTA Section**: Gradient button with glow effect

### 2. Device Repair Page
- Hero with AI repair lab imagery
- Service sections: Phone, Computer/Laptop, TV, Software fixes
- FAQ accordion
- Repair booking form

### 3. Networking & Security Page
- Hero with server rack/security imagery
- Sections: Network setup, Business WiFi, CCTV installation, Maintenance
- Consultation request form

### 4. Refurbished Products Page
- Hero with luxury device presentation
- Quality testing process, warranty info, environmental impact
- Product categories grid

### 5. About Us Page
- Mission, Vision, Team, Quality commitment
- AI-generated team and office imagery

### 6. Contact Page
- Contact form, phone, email, business hours
- Embedded map placeholder
- Live chat button

## Navigation
- Sticky premium navbar with logo, page links, and CTA button
- Mobile hamburger menu with slide-in drawer
- Smooth scroll and page transitions

## Animations
- Fade-in on scroll reveal for all sections
- Subtle parallax on hero images
- Hover lift/glow on cards and buttons
- Smooth page transitions between routes
